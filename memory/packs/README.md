# Memory Packs

Memory packs seed a project with a design personality. They are starting points, not
brand substitutes.

## Packs

- [Premium SaaS](premium-saas.md)
- [Enterprise Fintech](enterprise-fintech.md)
- [Cyber Dark](cyber-dark.md)
- [Playful Consumer](playful-consumer.md)

## Usage

Copy the chosen pack into `memory/design.md` or ask the agent to merge it:

```text
Use DesignOS memory pack `DesignOS/memory/packs/cyber-dark.md` as the initial design
personality. Merge it with the project brief and record final decisions in memory/design.md.
```
