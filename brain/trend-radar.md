# Trend Radar — Currency Without Cosplay

The Modernity dimension (`scoring/rubric.md`) demands work that reads *current-year
professional*. This module is the radar: what signals fresh, what signals dated, what
signals template — and the discipline for telling trend from principle.

> **Maintenance protocol:** this file has a shelf life by design.
> - **When:** quarterly (or after any major industry shift — a new platform, a viral design trend, a tool discontinuation)
> - **Who:** any contributor with evidence of the shift (link to real product/article, not opinion)
> - **How:** edit the status on existing entries (CURRENT → DATED, etc.) or add new entries to the CURRENT shelf with a date stamp. PR format: `chore(trend-radar): update <entry> status to DATED — [reason]`
> - **Review loop reference:** the quarterly health check in `loops/review-loop.md` includes re-dating this radar

## The three shelves (sort every visual idea)
```
PRINCIPLES   never expire: hierarchy, whitespace, type scales, contrast, honest states
             → the rest of DesignOS; trends never override these
CURRENT      credible now, will date: adopt deliberately, one per surface
DATED        signals neglect or template: avoid, or revive knowingly as retro (rare, expert)
```

## Current shelf (mid-2020s, status: alive)
- **Bento grids** for feature breadth — alive but commoditizing fast; execution quality
  now separates users from templates (`foundations/grids.md` rules).
- **Sticky product tours** (copy scrolls, product persists) — the workflow-story workhorse
  (`motion/page-and-scroll.md`).
- **Editorial-left oversized type heroes** — confidence signal, strongest in monochrome
  (`brain/reference-library.md` Vercel).
- **Warm-neutral "anti-SaaS" palettes + serif display** — the differentiation lane while
  blue-SaaS saturates (`psychology/color-psychology.md`).
- **⌘K palettes, keyboard-first signaling** — product-seriousness cue
  (`components/search-command.md`).
- **Dark-first dev aesthetics with mono textures** — stable in its sectors
  (`industries/` licensing).
- **Border-hairline structure over shadow structure** — flat-but-layered; shadows
  retreating to overlays only.
- **Real-product-as-hero** (screenshots, live demos) over abstract illustration —
  trust-driven, likely migrating to "principle" shelf.

## Dated shelf (signals neglect — the reviewer flags these)
- Purple-blue gradient meshes on AI anything (`industries/ai-startup.md` — the genre's
  stock photo)
- Corporate-Memphis flat humans; 3D blob mascots
- Glassmorphism everywhere (survives only as subtle nav blur); heavy neumorphism (died
  of contrast failure)
- Uppercase-letterspaced ghost buttons; Material ripples outside Material
- Carousel heroes; boxed "web 2.0" three-column icon features
- Parallax storytelling sites (outside expert editorial license)
- Skill bars, circular progress vanity stats (`industries/portfolio-agency.md`)
- Chat-bubble landing pages; typing-indicator theater on static content

## Watch list (emerging — adopt with eyes open)
- AI-native UI conventions (streaming output, generative canvases, chat-adjacent
  surfaces) — conventions still forming; follow the capability-honesty doctrine
  (`industries/ai-startup.md`) over the aesthetic
- Motion-rich scroll-driven CSS (`animation-timeline`) — capability outrunning taste;
  the budget rules (`motion/page-and-scroll.md`) matter more as the tech gets cheap
- Variable-font expressiveness (weight/width animation) — one signature use, not a circus
- P3/wide-gamut color moments — subtle depth advantage on modern screens, with sRGB
  fallback discipline

## The adoption discipline
1. **Trend adoption is a decision with a reason** — recorded in `memory/design.md` like
   any override: which trend, why it serves THIS brand/audience, where its single
   deployment lives.
2. **One current-shelf trend per surface** as the freshness signal; stacking three reads
   as template-chasing (the "2024 AI startup" uniform was exactly this stack).
3. **Sector gates trends** (`industries/` files): fintech adopts 18 months late by
   design; gaming adopts early by license.
4. **The revive rule:** dated elements return only as knowing retro-craft (brutalism,
   Y2K) — expert license, portfolio/editorial sectors, executed at 100% commitment or 0%.

## Calibration sources (how the radar stays honest)
Reference-canon shifts (`brain/reference-library.md` — when Stripe/Linear move, note
what and why), current award circuits (study, never copy — `industries/portfolio-agency.md`
warning), and the strongest signal: what premium products QUIETLY REMOVE (deletions by
the canon are trend obituaries).
