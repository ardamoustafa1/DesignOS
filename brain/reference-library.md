# Reference Library — The Canon

Named references the whole system grades against. Each entry: what they're the benchmark for,
and the *stealable tactics* — moves you can apply without copying pixels.

## Stripe — trust through craft
**Benchmark for:** marketing that converts skeptical technical buyers; information density done elegantly.
**Stealable tactics:**
- Claims are demonstrated, not asserted: live code blocks, real dashboards, interactive demos.
- The gradient is a *signature*, used once per page at most, in the hero mesh — never on buttons.
- Footer as a sitemap: comprehensive, organized, typographically quiet. Big footers signal big companies.
- Body copy reads like documentation: specific nouns, numbers, and API names instead of adjectives.
- Section rhythm: alternating proof types (visual demo → customer logos → metric → quote).

## Linear — speed as identity
**Benchmark for:** dark UI, product-led marketing, motion discipline.
**Stealable tactics:**
- Dark theme built from layered dark grays (#0A0A0B base, elevation via lighter surfaces), never pure black surfaces with pure white text.
- Animations 120–200ms, ease-out, mostly opacity+transform; the *product feels fast because the site is fast*.
- Marketing screenshots are the actual product, pixel-perfect, slightly angled or in a browser chrome.
- Keyboard shortcuts surfaced in marketing — signaling who the product is for.
- Release notes as a marketing channel: momentum is a feature.

## Apple — subtraction and scale
**Benchmark for:** hierarchy, whitespace, one-idea-per-viewport.
**Stealable tactics:**
- Each scroll viewport makes exactly one claim, with one supporting visual.
- Type scale jumps are dramatic (80px display next to 17px body) — hierarchy through contrast, not clutter.
- Product images enormous and impeccably lit; whitespace around them is the luxury signal.
- Copy rhythm: short declarative sentence. Then a supporting clause. Then silence.

## Vercel — monochrome authority
**Benchmark for:** developer credibility, doing more with black/white/one-accent.
**Stealable tactics:**
- Near-total monochrome; the rare accent (deploy-blue, gradient text once) lands with force.
- Borders at ~8% white opacity structure the entire dark UI — no shadows needed.
- Triangles/geometric brand motifs used structurally, not decoratively.
- Text hierarchy via weight+size alone proves the type system is sound.

## Notion — approachable competence
**Benchmark for:** friendly SaaS, illustration systems, template-led growth.
**Stealable tactics:**
- Monochrome line illustrations with a single accent — personality without visual noise.
- Feature explanations via real, recognizable use-cases ("meeting notes", not "knowledge synergy").
- Serif headlines + sans body as a warmth signal in a sea of grotesque-only SaaS.

## Airbnb — photographic trust
**Benchmark for:** marketplaces, search-first IA, social proof.
**Stealable tactics:**
- The search box *is* the hero. When the primary job is search, don't bury it under a slogan.
- Photography does the persuasion; UI recedes to white + one coral accent.
- Reviews with faces, names, dates — specificity is what makes social proof work.

## Figma — playful precision
**Benchmark for:** creative-tool energy that stays professional; multiplayer as a visual theme.
**Stealable tactics:**
- Cursor-with-name-tags as a brand device: the *feature* became the iconography.
- Bright color used in contained shapes on neutral ground — energy without chaos.

## Shopify — commerce pragmatism
**Benchmark for:** e-commerce patterns, merchant dashboards, conversion flows.
**Stealable tactics:**
- Checkout patterns treated as sacred: recognized layouts, zero innovation where money moves.
- Admin UI: dense tables + generous row height + sticky actions — density and calm coexisting.

## Usage rule

When the kernel says "how would X do this?", consult this file and answer with a *tactic*,
not a style transfer. DesignOS output should look like the client's brand executed at these
companies' standard — never like a clone of any of them.
