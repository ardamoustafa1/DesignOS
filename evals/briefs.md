# Eval Briefs

Ten briefs, fixed wording — run verbatim in both arms (`evals/README.md` protocol).
They span sectors, surface types, theme demands, and difficulty tiers. Output request
for all: *"Deliver a single self-contained HTML file."*

---

**B1 · SaaS landing (baseline)**
Design a landing page for "Ledgerline", a bookkeeping automation tool for small
agencies. Audience: agency owners who hate spreadsheets. Goal: trial signups.
We have no screenshots yet. Light theme.

**B2 · Dev-tool landing (dark, technical)**
Design a landing page for "Tracewise", a distributed-tracing tool for backend teams.
Audience: senior engineers, skeptical of marketing. Goal: docs visits and CLI installs.
Dark theme.

**B3 · Pricing page (persuasion mechanics)**
Design the pricing page for "Boardroom", a board-meeting management SaaS. Three tiers
(Starter, Growth, Enterprise), annual/monthly billing, 14-day trial, SOC2-sensitive
buyers. Include an FAQ.

**B4 · Dashboard (app UI, data density)**
Design the overview dashboard for "Fleetpulse", a delivery-fleet monitoring tool:
active vehicles, exceptions needing attention, on-time trend, and a recent-deliveries
table. Operators check it hourly. Dark theme.

**B5 · Checkout flow (money path, conventions)**
Design the cart-and-checkout for "Terra", a premium plant e-commerce shop. Guest
checkout, express pay options, shipping costs visible early. Light, warm.

**B6 · Healthcare booking (stressed users, trust)**
Design an appointment-booking page for "Nova Clinic", a private dermatology clinic.
Audience includes older patients. Needs: doctor selection, time slots, and reassurance
about data privacy. Calm, light.

**B7 · Portfolio (distinctiveness test)**
Design a portfolio home page for Mara Lin, a motion designer targeting music-industry
clients. 6 projects, one line of positioning, contact. She wants it to feel like HERS —
not like every portfolio.

**B8 · Fintech marketing (trust + numbers)**
Design a landing page for "Vaultic", a business savings product offering 4.2% APY,
regulated, FDIC-insured through partner banks. Audience: startup CFOs. Include honest
fine print handling.

**B9 · Docs quickstart (technical content)**
Design a documentation quickstart page for "Hookline", a webhooks-as-a-service API:
three-panel docs layout, install → configure → test steps with copyable code blocks.
Dark theme.

**B10 · Nonprofit donation (emotion + ethics)**
Design a donation page for "Deep Roots", a reforestation nonprofit. Preset amounts
with impact framing, monthly-first, one field-story, transparency numbers. No guilt
mechanics.

---

## Difficulty notes for interpreters
B1/B2 test baseline craft · B3/B5/B8 test convention adherence + honesty under
persuasion pressure · B4/B9 test app-grade density and technical content · B6/B10 test
register control (calm, ethics) · B7 is the distinctiveness stress test — expect the
smallest treatment gains there, and read `brain/originality.md` before over-interpreting.
