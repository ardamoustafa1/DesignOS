# Industry: AI Startup

## Buyer psychology
Audiences oscillate between **excitement** ("this changes everything") and **skepticism**
("this is a GPT wrapper / it hallucinates / where does my data go?"). The design's job is
to convert hype into *demonstrated capability*: show the model working on realistic input,
in the page if possible. Every unsubstantiated superlative deepens skepticism — the sector
has burned buyers before.

## Visual language
- **Color:** the purple-blue gradient mesh is the genre's stock photo — using it files you under "template." Differentiators that currently read as premium: monochrome + one surgical accent (Vercel-style), warm editorial neutrals + serif display, deep non-purple darks (petrol, forest, ink). If gradient: off-cliché axis, once (`foundations/colors.md`).
- **Type:** technical grotesque; mono accents for input/output, tokens, code — mono type is the "real AI" texture when showing actual model I/O.
- **Imagery:** live demo > recorded demo > annotated screenshot > abstract orb (last resort). Show INPUT → OUTPUT concretely; the transformation is the product.
- **Dark-first** works for infra/dev AI; consumer AI assistants trend light + friendly.

## Page priorities
1. Landing with an **interactive or auto-playing real demo** above the fold
2. Use-case pages per persona (the "what do I do with it" gap is the #1 AI bounce cause)
3. Pricing with usage calculator (`patterns/pricing.md` usage-based)
4. Trust/data page: training-data policy, retention, SOC2, "your data is never trained on" — in plain language, linked from the footer AND the signup form
5. Docs/API (if dev-facing) — docs quality is the buying signal

## Trust requirements
Concrete capability boundaries stated ("great at X, not built for Y" — honesty converts the
burned) · latency/accuracy numbers with methodology · data-handling one-liner at every input
field · named-customer proof over logo walls (AI logos are cheap; results aren't).

## Motion character
Streaming/typing output animation is the genre's licensed signature — use it for real output
only, with instant-complete on click and reduced-motion fallback. Otherwise precise preset;
the "everything glows and floats" AI look reads as compensation.

## Anti-patterns
Orb/brain/neural-mesh hero art · "Powered by AI" as the value prop (what does it DO?) ·
fake typing demos of canned text presented as live · benchmark charts with unlabeled axes ·
sparkle-emoji buttons everywhere · claiming AGI-adjacent anything.
