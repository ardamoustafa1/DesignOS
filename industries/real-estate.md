# Industry: Real Estate

## Buyer psychology
The largest purchase of a user's life, researched at night, emotionally. Two parallel modes:
**dreaming** (photo-led browsing, "what if") and **diligence** (numbers, neighborhoods,
commute, fees). Listings sites win the dream; the diligence layer wins the contact form.
For agencies/developers: the brand IS the trust — users are choosing whom to hand a fortune.

## Visual language
- **Photography is the interface:** full-bleed hero images, listing cards ≥70% photo, professional shots only (one dark phone-photo poisons a whole listing grid). Consistent aspect ratios (4:3 or 3:2) — mixed ratios wreck grid craft.
- **Color:** neutral, warm, receding — charcoal/warm-gray text, cream/white surfaces, one confident accent (deep green, navy, terracotta by brand). Luxury segment: near-black + gold-restraint + serif.
- **Type:** serif display (estate gravitas) + sans UI is the sector's native pairing; prices large and tabular; area units localized (m²/sqft).
- **Theme:** light; photography needs daylight.

## Page priorities & patterns
1. **Search-first hero:** the search box (location + type + price range) IS the hero (`brain/reference-library.md` Airbnb doctrine); background = one stunning property photo with scrim discipline
2. **Listing cards:** photo (with count badge + save-heart) → price (bold, first) → beds/baths/m² icon row → address → one status chip (New/Open house). Map+list split view with synced hover — the sector's power pattern
3. **Listing detail:** gallery first (grid-teaser → lightbox, keyboard-navigable) → key facts row → price + mortgage estimate widget → description → features → map + neighborhood layer (schools, transit) → agent card (photo, name, phone, one contact form) — sticky contact CTA on mobile
4. **Filters:** the product's core UX — price sliders with histograms, instant result counts, saved searches with alerts (the retention engine)
5. **Agent/agency pages:** real faces, real sales history, reviews — the human trust layer

## Trust requirements
Real photos (declare renders/staging on new builds) · full fee transparency (commissions,
dues) · license numbers where regulated · sold-price history where legal · verified badges
explained · response-time expectations on contact forms ("Agent replies within 2h").

## Motion character
Confident/premium preset: slow crossfade galleries, gentle parallax license on hero photography
(`motion/page-and-scroll.md` limits), map/list transitions smooth. No urgency animation —
"12 people viewed today" pulsing = instant sleaze.

## Anti-patterns
Watermark-soup photos · "$X,XXX,XXX" hidden prices ("contact for price" outside true luxury
kills trust) · fake urgency on homes · autoplay drone-video heroes with sound · gallery
lightboxes trapping keyboard/back-button · lead forms demanding phone number before showing
anything.

## References
