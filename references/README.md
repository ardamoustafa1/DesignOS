# Visual Reference Packs

Reference packs teach taste without copying. Load them when a brief asks for a visual
bar such as "Stripe-level," "Linear-like," "premium SaaS," or "cybersecurity dark UI."

## Packs

- [Stripe-Level Marketing](stripe-level-marketing.md)
- [Linear-Style App UI](linear-style-app.md)

- [Apple Product Page](apple-product-page.md)
- [Cybersecurity Dark UI](cybersecurity-dark-ui.md)

## Rules

- Extract rhythm, hierarchy, restraint, and proof strategy.
- Do not clone layout, copy, logos, gradients, or brand-specific motifs.
- Combine with `brain/originality.md` before final delivery.
- Score distinctiveness; "looks like X" is a failure unless the user asked for parody.
