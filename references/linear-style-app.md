# Reference Pack: Linear-Style App UI

Use for operational SaaS, project management, admin tools, and keyboard-heavy apps.

## Taste Signature

- Quiet chrome, high information density, minimal decoration.
- Keyboard-first interaction model.
- Crisp status language and fast transitions.
- Small details: focus rings, hover rows, command palette, stable layout.

## Transferable Moves

- Reduce marketing weight inside product surfaces.
- Favor scan structure over cards everywhere.
- Make the active object unmistakable.
- Use compact motion that confirms, never performs.
- Put expert shortcuts behind discoverable controls.

## Do Not Copy

- Exact sidebar composition.
- Purple/gray palette by default.
- Decorative blur fields.
- Hidden hover-only controls without focus/touch equivalents.

## Checklist

- [ ] Primary work object is obvious
- [ ] Repeated workflows require fewer clicks
- [ ] Dense tables stay readable
- [ ] Empty/loading/error states are first-class
- [ ] Command palette or search has keyboard contract
