# Starter: Mobile App

Use this for React Native, Flutter, native iOS, native Android, and mobile-first web.

## First Brief

```text
Design the first-run mobile experience for this app. Load DesignOS native/app-patterns.md,
native/ios.md, native/android.md, native/motion-gestures.md, patterns/onboarding-auth.md,
and psychology/habit-retention.md. Include permission timing, offline/interruption
recovery, reduced motion, and platform-native navigation. End with a scorecard and redo
anything under 95.
```

## Required Screens

- First value screen
- Sign-in or account creation
- Permission prompt moment
- Empty/offline state
- First success moment

## Verification

Run `designos review` against the generated screen files, then do a manual device pass
for touch target size and platform conventions.
