# Starter: Landing Page

Use this for one-page product launches, campaigns, waitlists, and validation pages.

## First Brief

```text
Design a conversion-focused landing page. Load DesignOS patterns/landing-pages.md,
components/hero.md, components/footer.md, psychology/persuasion.md, psychology/trust.md,
and the relevant industry module. Use only real proof. If proof is unavailable, show
product workflow instead. End with a scorecard and redo anything under 95.
```

## Required Sections

- Hero with product proof
- Problem / audience fit
- Workflow or feature proof
- Trust and objection handling
- CTA
- Footer

## Verification

```bash
node DesignOS/bin/designos.js review index.html --no-fail
```
