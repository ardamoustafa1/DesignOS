# Pre-Flight Checklist

The final gate before any deliverable is declared done. Binary items — a box either checks
or the work isn't finished. Run AFTER the specialized checklists (a11y, performance, seo,
responsive); this one catches what falls between them.

## Content integrity
- [ ] Zero lorem ipsum, `#` hrefs, "TODO", or placeholder blocks anywhere
- [ ] All numbers/claims real — or loudly flagged as placeholder in `memory/todo.md`
- [ ] Names, logos, testimonials verified real (or marked)
- [ ] Spelling/grammar pass done (typos are trust leaks — `psychology/persuasion.md`)
- [ ] Straight quotes replaced with real punctuation (’ “ ”); © year current
- [ ] One term per concept product-wide (glossary check — `agents/copywriter.md`)

## Every interactive element
- [ ] Hover, active, focus-visible, disabled states exist and look intentional
- [ ] Loading state where any async happens (width-locked buttons, skeletons)
- [ ] Cursor honest (`pointer` on interactive, nothing else)
- [ ] Destructive actions confirmed or undoable — never bare

## Every data surface
- [ ] State matrix complete: loading / empty (×3 species) / partial / error / success
- [ ] Error messages: cause + fix, no raw codes alone
- [ ] User work preserved through every failure path

## The chrome nobody remembers
- [ ] Favicon (all sizes) + og:image (1200×630, designed) + page titles unique
- [ ] 404 page designed; selection color set; scrollbar considered
- [ ] Print isn't broken (one glance — invoices/articles deserve a real pass)
- [ ] Dark/light: both deliberate, or single theme recorded as a decision

## System discipline
- [ ] Grep clean: no raw hex outside tokens, no px outside scales, no rogue z-index
- [ ] One icon set, one radius language, one shadow system, one motion character
- [ ] Fonts: only used weights loaded, `font-display: swap`, fallbacks metric-matched

## The three timescales (`brain/quality-bar.md`)
- [ ] Squint: structure legible blurred
- [ ] Glance (3s): what-is-this + what-do-I-do answered
- [ ] Minute: a skeptic finds real proof and no claim/craft contradiction

## Process closure
- [ ] All routed module checklists ran clean
- [ ] Scorecard filled (`scoring/report-template.md`), all dimensions ≥95
- [ ] Memory updated: design.md, pages.md, todo.md, bugs.md, notes.md
- [ ] Rationale written (3–6 decisions with module citations — kernel Output Contract)
