# Product Hunt

## Tagline

Give your AI coding agent design intelligence.

## Short description

DesignOS is an open-source, markdown-native design brain for Claude, Cursor, Copilot,
Windsurf, Cline, and Aider. It teaches agents design process, sector judgment, project
memory, and adversarial self-review before they ship UI.

## Maker comment

AI agents can now build entire apps, but their UI judgment still fails in boring,
expensive ways: weak hierarchy, invented testimonials, inaccessible contrast, and
decisions that drift from one session to the next.

DesignOS is my attempt to make design judgment inspectable and version-controlled.
It is not a UI kit. It is a design operating layer made of markdown modules, industry
playbooks, memory files, specialist reviewer agents, validators, and a hard 95/100
quality gate.

The repo is deliberately honest about what is proven and what is not. The current evals
are maintainer-run checks that prove the instrumentation fires; the big open milestone
is an independent Run 003 from outside users.

I would love feedback from builders using Claude Code, Cursor, Copilot, Windsurf, Cline,
or Aider.
